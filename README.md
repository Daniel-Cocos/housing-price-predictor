# Housing Price Predictor

Exploring the Ames Housing dataset through Exploratory Data Analysis. 

The objective is to identify the key factors influencing house prices and prepare the dataset for machine learning models.

---

## Features

- Data cleaning and preprocessing
- Missing value analysis
- Outlier detection
- Exploratory Data Analysis (EDA)

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

- Dataset is not included in the repository
- [Download dataset](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques) from Kaggle and place train.csv & test.csv inside data/raw/

```zsh
data/raw/
├── train.csv
└── test.csv
```
